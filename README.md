# 🛒 ShopSphere - MERN E-Commerce Application

## 📌 Live Demo
https://ecommerce-app-1-eq10.onrender.com

## 📖 Project Overview
ShopSphere is a full-stack e-commerce web application built using the MERN stack. It allows users to browse products, manage their cart, place orders, and track their purchases. The application also includes authentication and admin functionalities for product management.

---

## 🚀 Features

### User Features
- User Registration & Login
- JWT Authentication
- Browse Products
- Search Products
- Add Products to Cart
- Update Cart Quantity
- Remove Products from Cart
- Place Orders
- View Order History
- Cancel Pending Orders
- Dark/Light Theme Toggle
